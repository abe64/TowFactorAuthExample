# SpringBoot + Vue.js で一般的なアプリケーション基盤

## [Spring Tool Suite4](https://spring.io/tools) での環境構築手順

### 事前に[Spring Tool Suite4](https://spring.io/tools)のサイトでSTS4をダウンロードしてください。

![STS4](https://github.com/abe64/TowFactorAuthExample/blob/develop/images/setup001.png)

### STS4を起動してGitパースペクティブに切り替えてクローン作ります

適当なワークスペースでSTS4を起動して、パースペクティブをGitに切り替えます。

![画像](https://github.com/abe64/TowFactorAuthExample/blob/develop/images/setup002.png)

「Gitリポジトリのクローン」を選択します。

![画像](https://github.com/abe64/TowFactorAuthExample/blob/develop/images/setup003.png)

ロケーション等を入力します。

![画像](https://github.com/abe64/TowFactorAuthExample/blob/develop/images/setup004.png)

入力内容はURIを入れれば他は基本自動で補完されます。

|URI|
|:---|
|https://github.com/abe64/TowFactorAuthExample.git|

後はデフォルトのまま「完了(F)」

***
### ワークスペースにプロジェクトをインポートします

![画像](https://github.com/abe64/TowFactorAuthExample/blob/develop/images/setup005.png)

上記の「作業ツリー」からインポートしたいプロジェクトを選択 (ここではtwofactorauthAPI) して右クリック。
メニューの中から 「Mavenプロジェクトのインポート...」を選択します。

![画像](https://github.com/abe64/TowFactorAuthExample/blob/develop/images/setup006.png)

パースペクティブをJavaに切り替えて暫く待つと完了します。

![画像](https://github.com/abe64/TowFactorAuthExample/blob/develop/images/setup007.png)
